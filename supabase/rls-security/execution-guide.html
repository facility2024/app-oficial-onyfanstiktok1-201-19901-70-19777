Executar Script 01 - Sistema de Roles

CRÍTICO
  1. Clique em "New query" no SQL Editor
  2. Copie o conteúdo de: supabase/rls-security/01-critical-role-system.sql
  3. Cole no editor
  4. Clique em "Run" (ou pressione F5)
  5. Aguarde a execução (10-30 segundos)
💡 Dica: Você pode abrir o arquivo .sql no seu editor de código e copiar todo o conteúdo de uma vez.

Verificar:

SELECT * FROM public.user_roles; SELECT public.has_role('00000000-0000-0000-0000-000000000000'::uuid, 'admin');
3

Criar Usuário Admin Inicial

CRÍTICO

Passo 3.1: Descobrir seu User ID

-- Execute no SQL Editor: SELECT auth.uid(); -- OU veja todos os usuários: SELECT id, email, created_at FROM auth.users ORDER BY created_at DESC;

Passo 3.2: Criar Admin (substitua SEU-USER-ID-AQUI pelo UUID copiado)

INSERT INTO public.user_roles (user_id, role) VALUES ('SEU-USER-ID-AQUI', 'admin') ON CONFLICT (user_id, role) DO NOTHING;
⚠️ IMPORTANTE: Substitua 'SEU-USER-ID-AQUI' pelo seu UUID real!
Exemplo: 'a1b2c3d4-e5f6-7890-abcd-ef1234567890'

Verificar:

SELECT * FROM public.user_roles WHERE role = 'admin'; SELECT public.has_role(auth.uid(), 'admin'); -- Deve retornar true
